Paper Cat의 장점

여러가지 장점이 많지만, 여러분이 가장 흥미를 느낄만한 장점 3가지를 선정해보았습니다.

  • Version Control 논문 버전 컨트롤 지원
  • 17000+ 영어 논문 분석
  • 번역기 연동 구글/네이버 번역기 연동

Paper Cat 은 word, LaTeX 문서의 버전 컨트롤을 지원합니다. 또한 Github 와 연동하여 동료들과 협업할 수 있습니다.
또한, 실제 기제된 수많은 논문들을 분석하여 어러분에게 해당 자리와 의미에서 최적의 단어를 추천합니다.
더불어, 해당 단어의 번역 결과와 브라우저를 킬 필요 없이 툴에서 번역기를 사용할 수 있습니다.
